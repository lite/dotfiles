Dot Files
====

These are config files to set up a system the way I like it.

Installation
----

  git clone git://github.com/lite/dotfiles ~/.dotfiles
  cd ~/.dotfiles
  ./install.sh

  curl -L https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh | sh
